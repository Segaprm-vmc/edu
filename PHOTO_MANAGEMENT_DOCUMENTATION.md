# 📸 Система управления фотографиями для моделей мотоциклов

## 🎯 Обзор

Была добавлена полнофункциональная система для загрузки и управления фотографиями моделей мотоциклов в админ панели VMC Moto.

## ✨ Новые возможности

### 🔧 Админ панель
- **Загрузка фотографий** - Drag & Drop или выбор файлов
- **Управление порядком** - Перетаскивание для изменения последовательности отображения
- **Основная фотография** - Установка главного изображения для карточки модели
- **Удаление фотографий** - Безопасное удаление с подтверждением
- **Превью в реальном времени** - Мгновенный просмотр загруженных изображений

### 🎨 Отображение на сайте
- **Галерея фотографий** - Красивое отображение всех фотографий модели
- **Навигация** - Стрелки и точки для переключения между фотографиями
- **Миниатюры** - Быстрый доступ к любой фотографии
- **Отзывчивый дизайн** - Адаптация под любые размеры экрана

## 🛠️ Как использовать

### В админ панели:

1. **Откройте редактирование модели**
   - Перейдите в Админ панель → Модели
   - Нажмите "Редактировать" на любой модели

2. **Найдите секцию "Фотографии модели"**
   - Расположена после описания модели
   - Доступна только для существующих моделей

3. **Загрузите фотографии**
   - Перетащите файлы в зону загрузки или нажмите для выбора
   - Поддерживаются форматы: JPG, PNG, WebP
   - Максимальный размер: 10MB на файл

4. **Управляйте фотографиями**
   - **Изменение порядка**: Перетащите фотографию в нужную позицию
   - **Основная фотография**: Нажмите "Сделать основной"
   - **Удаление**: Нажмите иконку корзины

### На публичной странице модели:

1. **Основное изображение**
   - Отображается главная фотография (или первая по порядку)
   - Навигация стрелками влево/вправо
   - Индикаторы точками внизу

2. **Миниатюры**
   - Сетка до 6 миниатюр под основным изображением
   - Клик по миниатюре переключает основное изображение
   - Желтая точка обозначает основную фотографию

## 🔧 Техническая реализация

### Backend (FastAPI)
```python
# Новые API endpoints:
POST   /admin/models/{model_id}/photos      # Загрузка фотографии
GET    /admin/models/{model_id}/photos      # Получение списка фотографий
PUT    /admin/models/{model_id}/photos/order # Изменение порядка
PUT    /admin/photos/{photo_id}/primary     # Установка основной
DELETE /admin/photos/{photo_id}             # Удаление фотографии
```

### Frontend (React)
```tsx
// Новые компоненты:
<PhotoManager />           // Управление фотографиями в админке
<ModelForm />             // Обновлен для работы с фотографиями
<ModelDetailPage />       // Обновлен для отображения галереи
```

## 🔄 Workflow для контент-менеджера

1. **Подготовка фотографий**
   - Качественные изображения (желательно 1920x1080 или выше)
   - Хорошее освещение
   - Разные ракурсы модели

2. **Загрузка в админку**
   - Сначала сохраните основную информацию о модели
   - Затем загрузите фотографии через PhotoManager
   - Установите самую показательную как основную

3. **Оптимизация порядка**
   - Основная фотография - общий вид модели
   - Далее - детали, особенности, ракурсы
   - Последние - дополнительные виды

## 🐛 Troubleshooting

### Частые проблемы

**Ошибка "Сначала сохраните модель"**
- Нужно создать модель и сохранить её
- Фотографии можно загружать только для существующих моделей

**Файл не загружается**
- Проверьте формат (JPG, PNG, WebP)
- Убедитесь, что размер не превышает 10MB
- Проверьте подключение к интернету

**Фотографии не отображаются на сайте**
- Убедитесь, что backend сервер запущен
- Проверьте права доступа к папке uploads
- Обновите страницу (Ctrl+F5)
