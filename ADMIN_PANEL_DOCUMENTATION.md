# 🛠️ Админ панель VMC Moto - Документация

## 🎯 **Обзор**

Создана полнофункциональная админ панель для управления моделями мотоциклов с чистым и современным интерфейсом.

## ✨ **Основные возможности**

### 1. **Управление моделями мотоциклов**
- ✅ Просмотр списка всех моделей
- ✅ Добавление новых моделей
- ✅ Редактирование существующих моделей
- ✅ Удаление моделей с подтверждением
- ✅ Поиск по моделям
- ✅ Фильтрация по категориям

### 2. **Управление характеристиками**
- ✅ Выбор модели для редактирования характеристик
- ✅ Добавление/редактирование/удаление технических характеристик
- ✅ Группировка характеристик по категориям
- ✅ Поиск по характеристикам
- ✅ Сортировка характеристик

### 3. **Аутентификация**
- ✅ Защищенная админ панель
- ✅ Форма входа с валидацией
- ✅ Сохранение состояния входа

## 🏗️ **Архитектура компонентов**

```
AdminPage.tsx          - Главная админ панель
├── ModelForm.tsx      - Форма добавления/редактирования моделей
├── SpecsManager.tsx   - Управление характеристиками
└── AdminLoginPage.tsx - Страница входа в админку
```

## 📁 **Структура файлов**

### **Страницы**
- `frontend/src/pages/AdminPage.tsx` - Основная админ панель
- `frontend/src/pages/AdminLoginPage.tsx` - Страница входа

### **Компоненты**
- `frontend/src/components/ModelForm.tsx` - Форма моделей
- `frontend/src/components/SpecsManager.tsx` - Управление характеристиками

## 🚀 **Запуск админ панели**

### 1. Запуск серверов
```bash
# Backend
cd backend
py main.py

# Frontend (в отдельном терминале)
cd frontend
npm run dev
```

### 2. Доступ к админ панели
- **URL:** http://localhost:3000/admin/login
- **Пароль:** `admin123`

## 🎨 **Интерфейс админ панели**

### **Навигация**
1. **Модели** - Список всех моделей мотоциклов
2. **Добавить модель** - Форма создания/редактирования
3. **Характеристики** - Управление техническими характеристиками

### **Функции управления моделями**
- **Поиск** - Поиск по названию и категории
- **Добавить** - Создание новой модели
- **Редактировать** - Изменение существующей модели
- **Удалить** - Удаление с подтверждением
- **Импорт/Экспорт** - Заготовки для массовых операций

- **Ключевые особенности*** - Массив строк
- **URL изображения** - Ссылка на изображение модели

## 🔧 **Управление характеристиками**

### **Категории характеристик:**
1. **Двигатель** - Тип, объем, мощность, охлаждение
2. **Шасси и подвеска** - Рама, передняя/задняя подвеска
3. **Тормозная система** - Передний/задний тормоз, ABS
4. **Колеса и шины** - Размеры колес и шин
5. **Габариты и масса** - Длина, ширина, высота, масса
6. **Электрооборудование** - Системы освещения, зарядки
7. **Характеристики производительности** - Скорость, ускорение
8. **Дополнительные особенности** - Специальные функции
9. **Прочие характеристики** - Остальные параметры

### **Поля характеристики:**
- **Название*** - Объем двигателя
- **Значение*** - 450
- **Единица измерения** - куб.см
- **Категория*** - Двигатель
- **Порядок сортировки** - Для упорядочивания

## 🔄 **Интеграция с Backend**

Компоненты готовы для интеграции с реальным API:

### **Необходимые API endpoints:**
```typescript
// Модели
GET    /api/admin/models          - Список моделей
POST   /api/admin/models          - Создание модели
PUT    /api/admin/models/{id}     - Обновление модели
DELETE /api/admin/models/{id}     - Удаление модели

// Характеристики
GET    /api/admin/models/{id}/specs     - Характеристики модели
POST   /api/admin/models/{id}/specs     - Добавление характеристики
PUT    /api/admin/specs/{id}            - Обновление характеристики
DELETE /api/admin/specs/{id}            - Удаление характеристики
```
