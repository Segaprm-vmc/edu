# Обновление: Системные категории по умолчанию

## ✅ Что добавлено

### 🏷️ Системные категории
Добавлены **5 основных категорий**, которые всегда доступны в админ-панели:

1. **Мотоциклы** 
2. **Эндуро**
3. **Скутеры** 
4. **Мопеды**
5. **Питбайки**

### 🔒 Защита системных категорий

- ✅ **Нельзя удалить** - системные категории защищены от удаления
- ✅ **Нельзя редактировать** - названия фиксированы и неизменны  
- ✅ **Всегда доступны** - автоматически загружаются при запуске админки
- ✅ **Визуальное отличие** - отмечены синим цветом и меткой "Системная"

### 🎯 Преимущества

**Для пользователей:**
- Стандартизированные категории всегда под рукой
- Нет риска случайного удаления важных категорий
- Консистентность данных

**Для системы:**
- Гарантированная целостность каталога
- Предсказуемая структура категорий
- Упрощенная навигация

### 🔧 Техническая реализация

#### Константы
```typescript
const DEFAULT_CATEGORY_IDS = ['motorcycles', 'enduro', 'scooters', 'mopeds', 'pitbikes'];
```

#### Защита от удаления
```typescript
if (DEFAULT_CATEGORY_IDS.includes(categoryId)) {
  alert('Системные категории нельзя удалить.');
  return;
}
```

#### Визуальное отличие
- **Системные:** Голубой фон + метка "Системная" + "Защищена"
- **Пользовательские:** Серый фон + кнопки редактирования/удаления


### ✨ Возможности

1. **Добавление моделей** - все 5 категорий всегда доступны в выпадающем списке
2. **Создание дополнительных категорий** - можно добавлять свои категории
3. **Автоматический подсчет** - количество моделей в каждой категории

### 🎮 Как использовать

1. **Добавление модели:**
   - Выберите одну из системных категорий
   - Или создайте новую пользовательскую категорию

2. **Управление категориями:**
   - Переходите на вкладку "Категории"
   - Системные категории отмечены синим
   - Добавляйте свои категории кнопкой "Добавить категорию"
