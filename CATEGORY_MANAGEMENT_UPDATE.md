## Интерфейс

### Вкладка "Категории"
- Отображение всех категорий в виде карточек
- Показ количества моделей в каждой категории
- Кнопки редактирования и удаления для каждой категории
- Кнопка "Добавить категорию" в заголовке

### Форма модели
- Выпадающий список с существующими категориями
- Кнопка "+ Новая категория" для быстрого добавления
- Инлайн форма с полем ввода и кнопками сохранения/отмены

## Технические детали

### Интерфейсы
```typescript
interface Category {
  id: string;
  name: string;
  count: number;
}
```

### Функции управления
- `handleAddCategory()` - добавление новой категории
- `handleEditCategory()` - редактирование категории
- `handleDeleteCategory()` - удаление с подтверждением
- `updateCategoryCounts()` - пересчет количества моделей

## Использование

1. **Управление категориями:**
   - Перейдите на вкладку "Категории" в админ-панели
   - Используйте "Добавить категорию" для создания новых
   - Кликните на иконки редактирования/удаления

2. **Создание модели с новой категорией:**
   - При добавлении модели нажмите "+ Новая категория"
   - Введите название и нажмите Enter или кнопку сохранения
   - Категория будет сразу доступна для выбора
