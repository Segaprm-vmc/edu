# 📊 Управление техническими характеристиками

## 🎯 Обзор

Новая система управления техническими характеристиками позволяет:
- ✅ Добавлять характеристики вручную
- ✅ Импортировать характеристики из Excel файлов
- ✅ Редактировать характеристики прямо в форме
- ✅ Удалять ненужные характеристики
- ✅ Скачивать шаблон Excel для правильного формата

## 📋 Структура характеристики

Каждая характеристика содержит:
- **Название характеристики** (обязательно) - например, "Объем двигателя"
- **Значение** (обязательно) - например, "450"
- **Единица измерения** (опционально) - например, "куб.см"
- **Категория** (опционально) - например, "engine"

## 📊 Формат Excel файла

### Структура столбцов:
| A | B | C | D |
|---|---|---|---|
| Название характеристики | Значение | Единица измерения | Категория |
| Объем двигателя | 450 | куб.см | engine |
| Максимальная мощность | 45 | л.с. | engine |
| Снаряженная масса | 165 | кг | dimensions |

### Требования к файлу:
1. **Формат**: `.xlsx` или `.xls`
2. **Первая строка**: заголовки (будет пропущена при импорте)
3. **Обязательные столбцы**: A (название) и B (значение)
4. **Опциональные столбцы**: C (единица измерения) и D (категория)

### Пример шаблона:
```
Название характеристики | Значение | Единица измерения | Категория
Объем двигателя        | 450      | куб.см           | engine
Максимальная мощность  | 45       | л.с.             | engine
Снаряженная масса      | 165      | кг               | dimensions
Длина                  | 2150     | мм               | dimensions
```

## 🚀 Как использовать

### 1. Добавление характеристик вручную
1. Перейдите в форму редактирования модели
2. Найдите секцию "Технические характеристики"
3. Нажмите кнопку "Добавить"
4. Заполните поля и сохраните

### 2. Импорт из Excel
1. Подготовьте Excel файл по указанному формату
2. В секции "Технические характеристики" нажмите "Выбрать файл"
3. Выберите Excel файл или перетащите его в зону загрузки
4. Характеристики автоматически добавятся к модели

### 3. Скачивание шаблона
1. В секции "Технические характеристики" нажмите "Скачать шаблон"
2. Откроется готовый Excel файл с примерами
3. Отредактируйте его под ваши характеристики
4. Загрузите обратно в систему

## 🔧 Технические особенности

### Frontend компонент: `SpecsExcelImport.tsx`
- **Drag & Drop**: Поддержка перетаскивания файлов
- **Валидация**: Проверка формата файлов
- **Парсинг**: Использует SheetJS (xlsx) для чтения Excel
- **UI**: Современный интерфейс с обратной связью

### API endpoints:
- `GET /models/{model_id}/specs` - получение характеристик
- `POST /admin/models/{model_id}/specs` - создание характеристики
- `PUT /admin/specs/{spec_id}` - обновление характеристики
- `DELETE /admin/specs/{spec_id}` - удаление характеристики
- `POST /admin/models/{model_id}/specs/import` - импорт из Excel

### Интеграция с ModelForm:
```typescript
import SpecsExcelImport from './SpecsExcelImport';
import { ModelSpec } from '../services/api';

// В компоненте:
<SpecsExcelImport
  modelId={model?.id ? parseInt(model.id) : undefined}
  specs={specs}
  onSpecsChange={setSpecs}
  onUploadSpecs={handleUploadSpecs}
  isLoading={isLoadingSpecs}
/>
```

## 🐛 Troubleshooting

### Частые проблемы:

**1. "Ошибка при чтении Excel файла"**
- Проверьте формат файла (.xlsx или .xls)
- Убедитесь что файл не поврежден
- Проверьте наличие данных в столбцах A и B

**2. "Характеристики не сохраняются"**
- Убедитесь что модель уже создана (имеет ID)
- Проверьте подключение к интернету
- Проверьте права доступа администратора

**3. "Дублирующиеся характеристики"**
- Система автоматически пропускает дубли по названию
- Для замены существующих характеристик удалите их сначала
