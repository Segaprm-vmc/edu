# 🚀 Инструкции по запуску VMC Moto Educational Portal

## Запуск Backend (FastAPI)

```bash
# Перейти в папку backend
cd backend

# Запустить сервер разработки  
py main.py

# Или альтернативно:
py -m uvicorn main:app --reload --host 0.0.0.0 --port 8000
```

**Backend будет доступен:**
- 🌐 Главная страница: http://localhost:8000
- 📖 API документация: http://localhost:8000/api/docs  
- 🔥 Redoc документация: http://localhost:8000/api/redoc

## Запуск Frontend (React)

```bash
# Перейти в папку frontend  
cd frontend

# Запустить dev сервер
npm run dev
```

**Frontend будет доступен:**
- 🎨 Приложение: http://localhost:3000

## 🔧 Настройки проекта

### База данных:
- 📊 **SQLite** (vmcmoto_edu.db) - автоматически создается
- 🔄 Миграции через **Alembic**

### Конфигурация:
- ⚙️ Backend настройки: `backend/.env`
- 🎯 Frontend настройки: `frontend/vite.config.ts`

## 🛠️ Разработка:

**Backend разработка:**
- FastAPI с автоматической генерацией документации
- SQLAlchemy ORM для работы с БД
- Pydantic для валидации данных

**Frontend разработка:**  
- React 18 + TypeScript
- Vite для быстрой сборки
- TailwindCSS для стилизации
