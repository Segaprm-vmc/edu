# VMC Moto Educational Portal

Образовательный портал VMC Moto - современное веб-приложение для изучения мототехники с удобным интерфейсом и системой фильтрации.

## 🚀 Технологии

### Frontend
- **React 18** + **TypeScript**
- **Vite** (быстрая сборка)
- **TailwindCSS** (стилизация)
- **Lucide React** (иконки)
- **React Router** (роутинг)

### Backend
- **FastAPI** (Python)
- **SQLAlchemy** (ORM)
- **Pydantic** (валидация данных)

## ✨ Возможности

- 🏍️ **Каталог мототехники** - 7 моделей в 5 категориях
- 🔍 **Умный поиск** - по названию, описанию и характеристикам
- 📂 **Фильтрация по категориям** через боковое меню
- 📱 **Адаптивный дизайн** - работает на всех устройствах
- 🎨 **Современный UI** - минималистичный и удобный

## 🛠️ Установка и запуск

### Frontend

```bash
# Переход в папку frontend
cd frontend

# Установка зависимостей
npm install

# Запуск dev сервера
npm run dev
```

Приложение будет доступно по адресу: `http://localhost:5173`
